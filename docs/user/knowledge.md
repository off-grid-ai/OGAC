# Knowledge

*Skeleton (how/what/why/when) — to be deepened.* Surface: **Workspace → Knowledge (`/knowledge`)**.

## What it is

Ask-Your-Org, on-prem: an admin-curated shared knowledge base, indexed once and retrieved permission-aware, with citations in chat.

## Why use it

- Ground the whole org's chat in trusted, curated docs — with citations, so answers are checkable.
- Permission-aware retrieval: people only ground on what they're allowed to see.

## When to use it

- When you want a single source of truth the assistant cites, not per-user uploads.
- For SOPs, policies, product docs the org should answer from.

## How to use it

Curate the knowledge base (add/update/remove sources); it's indexed for retrieval. Chat and agents ground on it with citations. Manage sources and re-index from this surface.
