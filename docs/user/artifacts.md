# Artifacts

*Skeleton (how/what/why/when) — to be deepened.* Surface: **Workspace → Artifacts (`/artifacts`, Workspace tabs)**.

## What it is

A library of generated outputs — HTML, SVG, React, diagrams, code — saved from your chats and reopenable anytime.

## Why use it

- Keep the useful things the assistant produced instead of losing them in chat scrollback.
- Reopen, copy, or iterate on a saved output later.

## When to use it

- Whenever a chat produces something worth keeping (a component, a diagram, a script).
- When you want a durable link to a generated output.

## How to use it

Save an artifact from a chat; browse, reopen, and delete them from the Artifacts library. Reached via the Workspace top-tabs (route restored in the workspace consolidation — it's reachable again).
