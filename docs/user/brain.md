# Brain

*Skeleton (how/what/why/when) — to be deepened.* Surface: **Intelligence → Brain (`/brain`)**.

## What it is

Ingestion → retrieval (RAG): the knowledge base, SOPs, and citations that ground answers. Vector-backed retrieval.

## Why use it

- Turn documents into grounded, cited answers.
- Retrieval is permission-aware and on-prem.

## When to use it

- When answers must cite source docs.
- When building an assistant over a corpus.

## How to use it

Ingest text / files / images / datasets, watch them index, and retrieve with citations. Manage the collections and re-index. See [Data & Retrieval](retrieval-knowledge.md) for the vector store view.
