# Prompts

*Skeleton (how/what/why/when) — to be deepened.* Surface: **Workspace → Prompts (`/prompts`, Workspace tabs)**.

## What it is

A library of reusable prompts — save, tag, and organize prompt texts — plus a Common Prompts view mined from what the org actually asks.

## Why use it

- Stop re-typing good prompts; standardize how the team asks.
- Discover what's actually being asked (Common Prompts) and promote the best into the library.

## When to use it

- When a prompt works well and you'll want it again.
- When onboarding a team to a repeatable workflow.

## How to use it

Create a prompt (title, body, tags), edit and delete from the library. Browse Common Prompts to see mined org usage. Reached via the Workspace top-tabs.
